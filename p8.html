Project Idea

I wanted to create a game that was about traveling through different sceneries, specifically dreamscapes, and then the ending would be waking up from that dream and returning to reality.

Reflection

I struggled a lot with trying to code a figure traveling through a space, and I couldnt' figure it out, so I ended up using a code that already existed and added my own images that I created in illustrator. I realize that a lot of coding is using tools that may already exist, and that can help you learn and grow in your own coding.
